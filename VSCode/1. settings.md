**注意**：使用前需删除所有注释

```json
{
    // 设置 Markdown 预览增强插件的主题为 atom-dark
    "markdown-preview-enhanced.previewTheme": "atom-dark.css",

    // 启用 Git 智能提交（跳过输入提交信息的确认）
    "git.enableSmartCommit": true,

    // 禁用同步前的确认提示
    "git.confirmSync": false,

    // 自动获取远程 Git 仓库的更新
    "git.autofetch": true,

    // 当编辑器焦点变化时自动保存文件
    "files.autoSave": "OnFocusChange",
}
```
